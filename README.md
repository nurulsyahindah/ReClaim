## ReClaim (Lost & Found)

ReClaim is a web-based lost and found system developed to help users report, search, and claim lost items.

## Features
### User

- Register and log in.
- Submit lost and found reports.
- Search for items.
- Claim items.
- View report history.
- Receive notifications.

**Admin**

- Manage reports.
- Approve or reject reports.
- Manage claims.
- Manage users.
- Manage categories.
- View notifications.

## Technologies Used

- HTML
- CSS
- Bootstrap 5
- JavaScript
- PHP
- CakePHP 5
- MySQL

## User Roles

### User

Users can:
- Submit reports.
- Search for items.
- Claim items.
- Track report status.

### Admin

Administrators can:
- Access the admin dashboard.
- Manage reports.
- Manage claims.
- Manage users.
- Manage categories.
